🎯 1. 介紹

批量檔案重命名工具是一個專為 Windows 用戶設計的桌面應用程式，旨在簡化批量檔案重命名的操作流程。無論您是需要整理照片檔案、規範文件命名，還是進行其他檔案管理任務，這個工具都能為您提供強大而靈活的解決方案。

多樣化規則

支援前綴/後綴添加、文字替換、序列編號、大小寫轉換等多種重命名方式，可組合使用以滿足複雜需求。

即時預覽

在執行重命名前查看結果預覽，自動檢測檔名衝突，確保操作的準確性和安全性。

操作記錄

完整記錄每次重命名操作，支援一鍵復原，讓您可以安心進行批量操作。

智能過濾

支援按副檔名、檔名模式過濾檔案，精確控制要處理的檔案範圍。

💾 2. 安裝與啟動

系統需求

  • 作業系統：Windows 7/8/10/11
  • Python 版本：3.7 或更高版本
  • 硬碟空間：至少 50MB 可用空間

安裝步驟

步驟 1： 確保您的系統已安裝 Python 3.7+。如果沒有，請前往 Python 官網 下載並安裝。
步驟 2： 下載專案檔案到您的電腦（例如桌面）。
步驟 3： 雙擊 run.bat 檔案啟動程式，或在命令提示字元中執行 python main.py。
💡 提示： 首次啟動可能需要幾秒鐘的初始化時間，這是正常現象。

🖥️ 3. 界面介紹

應用程式採用直觀的分頁式設計，主要包含以下區域：

主要區域

  • 目錄選擇欄： 用於選擇要處理的檔案目錄
  • 規則設定面板： 配置各種重命名規則和檔案過濾條件
  • 預覽面板： 顯示重命名結果預覽和檔案詳細資訊
  • 操作歷史： 查看和管理過往的重命名操作記錄
  • 狀態列： 顯示當前操作狀態和檔案統計資訊

工具列按鈕

  • 瀏覽目錄 - 選擇要處理的檔案目錄
  • 預覽重命名 - 生成重命名結果預覽
  • 執行重命名 - 執行實際的重命名操作
  • 復原操作 - 復原上一次的重命名操作
  • 刷新列表 - 重新載入檔案列表

🚀 4. 基本使用方法

快速入門

1. 選擇目錄
點擊「瀏覽目錄」按鈕，選擇包含要重命名檔案的資料夾。
2. 設定過濾條件（可選）
在「檔案過濾」區域輸入副檔名，例如：.txt,.jpg,.png
3. 配置重命名規則
選擇規則類型（前綴、後綴、替換等），填入相應參數，點擊「應用規則」。
4. 預覽結果
點擊「預覽重命名」查看結果，檢查是否有衝突或錯誤。
5. 執行重命名
確認無誤後點擊「執行重命名」，系統會提示確認對話框。
⚠️ 重要提醒： 在執行重命名前，建議先備份重要檔案，或在少量檔案上測試規則的正確性。

📝 5. 重命名規則詳解

前綴/後綴

前綴添加

功能： 在檔名前方添加指定文字

範例： document.txt → 新_document.txt

用途： 為檔案添加分類標籤、日期前綴等

後綴添加

功能： 在檔名後方（副檔名前）添加指定文字

範例： document.txt → document_備份.txt

用途： 添加版本號、狀態標記等

文字替換

字串替換

功能： 將檔名中的特定文字替換為新文字

範例： old_file.txt → new_file.txt（將 "old" 替換為 "new"）

選項：

  • 「包含副檔名」：是否在副檔名中也進行替換
  • 支援區分大小寫的替換

序列編號

自動編號

功能： 為檔案添加自動遞增的序列編號

設定選項：

  • 起始編號： 編號的起始數字（如：1、100）
  • 位數： 編號的總位數，不足補零（如：3位數產生 001、002、003）

範例： photo1.jpg, photo2.jpg → 001.jpg, 002.jpg

大小寫轉換

大小寫控制

選項說明：

  • 保持原樣： 不改變大小寫
  • 全部大寫： filename.txt → FILENAME.txt
  • 全部小寫： FILENAME.txt → filename.txt
  • 首字母大寫： filename.txt → Filename.txt
  • 標題格式： file name.txt → File Name.txt

規則組合

💡 進階技巧： 您可以同時應用多個規則，系統會按照添加順序依次執行。例如：先進行文字替換，再添加前綴，最後轉換大小寫。

⚙️ 6. 進階功能

檔案過濾

檔案過濾功能讓您可以精確控制要處理的檔案範圍：

  • 副檔名過濾： 輸入 .txt,.jpg,.png 只處理這些類型的檔案
  • 空白過濾： 留空則處理目錄中的所有檔案
  • 多重條件： 用逗號分隔多個副檔名

預覽與檢測

預覽功能提供詳細的重命名資訊：

  • 即時預覽： 顯示重命名前後的檔名對比
  • 衝突檢測： 自動標記檔名衝突和無效檔名
  • 統計資訊： 顯示總檔案數、變更數、衝突數
  • 檔案詳情： 查看檔案大小、修改時間等資訊

操作歷史

歷史功能為您提供完整的操作追蹤：

  • 完整記錄： 記錄每次重命名的詳細資訊
  • 一鍵復原： 可復原任意歷史操作
  • 歷史管理： 查看、刪除、匯出操作記錄
  • 自動清理： 自動保留最近的操作記錄

🛡️ 7. 安全提醒

重要安全建議

  • 備份重要檔案： 在進行大量重命名前，請先備份重要資料
  • 小範圍測試： 建議先在少量檔案上測試規則的正確性
  • 仔細檢查預覽： 務必仔細檢查預覽結果再執行重命名
  • 避免系統檔案： 不要在系統重要目錄中使用此工具
  • 檔案權限： 確保對目標檔案有足夠的讀寫權限

系統限制

  • 檔名長度受 Windows 系統限制（通常為 255 字元）
  • 無法重命名正在被其他程式使用的檔案
  • 某些系統檔案和受保護檔案無法重命名
  • 不支援跨磁碟機的檔案移動操作

🔧 8. 疑難排解

常見問題

問題：程式無法啟動

可能原因：

  • Python 版本過舊或未安裝
  • tkinter 模組缺失
  • 檔案權限問題

解決方法：

  • 確認 Python 版本為 3.7 或更高
  • 重新安裝 Python（確保包含 tkinter）
  • 以系統管理員身分執行

問題：重命名操作失敗

可能原因：

  • 檔案被其他程式佔用
  • 沒有足夠的寫入權限
  • 檔名包含無效字元
  • 目標檔名已存在

解決方法：

  • 關閉可能使用檔案的程式
  • 檢查檔案和目錄權限
  • 檢查重命名規則是否產生有效檔名
  • 使用預覽功能檢查衝突

問題：預覽結果不正確

可能原因：

  • 規則設定錯誤
  • 檔案過濾條件有誤
  • 檔案列表需要刷新

解決方法：

  • 檢查並調整重命名規則
  • 確認檔案過濾設定
  • 點擊「刷新列表」按鈕
  • 重新選擇目錄

效能優化建議

  • 對於大量檔案（超過 1000 個），建議分批處理
  • 避免在網路磁碟機上直接操作
  • 處理大檔案時可能需要更多時間，請耐心等待
  • 定期清理操作歷史以節省儲存空間
